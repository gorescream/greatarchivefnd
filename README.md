# greatarchivefnd
this tool can extract big files from the zip by the chunks and (or) find the string in the file on archive, without fill unpacking on the disk
In the ex.py use archive_path, file_inside (inside the archive) and output_path for settings. then start ex.py. file on the output_path let rewrited for every chunk, if ypu need save chunk just copy file.
In the find.py use FILE_INSIDE, FILE_INSIDE, SEARCH_STRING and ENCODING for settings, then start find.py
